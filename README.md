# Statistical Distribution Analysis using Python

## Project Overview

This project focuses on statistical distribution analysis and probability modeling using Python. The dataset contains transaction-related information such as transaction amount, transaction count, transaction status, region, and customer details.

The project applies different probability distributions and statistical techniques to analyze transaction behavior and visualize data patterns.

# Conclusion

The analysis showed that transaction amounts are positively skewed and better represented using Log-Normal Distribution. Poisson Distribution effectively modeled transaction counts, while Box-Cox Transformation improved data stability and normality.
